Projects

Explore my projects.

Panopticon

Project Panopticon is a real-time, browser-based Open Source Intelligence (OSINT) and Threat Intelligence framework. It provides security analysts and investigators with a highly visual, terminal-style interface to track IP reputations, domain history, cryptocurrency traces, and dark web data breaches—all from a single, lightweight dashboard.

WhaleWatcher

Whale Watcher is a highly visual, real-time cryptocurrency surveillance dashboard designed to track massive financial movements across the blockchain. Built with a premium, dark-mode Glassmorphism aesthetic, the application acts as a live "ticker tape" for high-value trades, separating market noise from institutional activity.Instead of relying on restricted or rate-limited third-party APIs, Whale Watcher connects directly to primary cryptocurrency exchange live aggTrade WebSocket streams, parsing continuous global transactions for major pairs (BTC, ETH, SOL, BNB, DOGE).

PhishGuard

PhishGuard is a high-performance, fully client-side cybersecurity dashboard designed to perform deep, real-time intelligence gathering on suspicious URLs and domains.Built with a premium neon-glassmorphism aesthetic, the engine relies entirely on keyless architecture—querying public Open-Source Intelligence (OSINT) networks and executing local heuristic analysis without the need for a backend server or database.

Caesar-Cipher

A cryptography tool that encrypts and decrypts text using the classic Caesar cipher technique. It allows for customizable shift values to secure text-based communications.

Image-Encryption

A secure utility designed to encrypt and decrypt image files using pixel manipulation and cryptographic algorithms, ensuring complete visual data privacy.
